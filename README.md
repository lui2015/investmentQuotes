# 💎 投资名言（InvestmentQuotes）

> 让大师思想照亮普通人的投资道路

一个精美的投资名言聚合网站，汇聚巴菲特、芒格、格雷厄姆等全球投资大师的经典名言，支持按大师和投资思路分类浏览，每日推荐一句智慧箴言。

## 快速启动

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

打开 [http://localhost:3000](http://localhost:3000) 即可访问。数据库会在首次启动时自动创建并填充种子数据。

## 技术栈

| 层级 | 技术方案 |

# 产品需求文档 (PRD)

## 一、项目概述

### 1.1 项目背景

投资是一场漫长的修行，巴菲特、芒格、格雷厄姆、彼得·林奇等投资大师用数十年实践凝练出的智慧，往往浓缩在一句话中。然而这些名言散落在各类书籍、演讲和访谈里，普通投资者难以系统获取。

**投资名言** 旨在打造一个精美、易用的名言聚合网站，让用户随时汲取大师智慧，在投资路上少走弯路。

### 1.2 目标用户

| 用户群体 | 特征 |
|---------|------|
| 投资新手 | 刚入市，需要建立正确的投资理念 |
| 个人投资者 | 有一定经验，希望系统性学习大师思想 |
| 理财博主/自媒体 | 需要高质量的投资金句作为内容素材 |
| 金融从业者 | 日常分享、培训中引用大师名言 |

### 1.3 项目目标

- 收录 **50+ 位投资大师**、**500+ 条精选名言**（初期）
- 提供多维度分类浏览与搜索能力
- 首页每日推荐，培养用户访问习惯
- 移动端优先，响应式设计

### 2.3 名言详情

| 大师链接 | 点击大师名字跳转大师详情页 |

### 2.4 名言深度解读 ✅ (v1.2 新增)

为避免"名言孤零零挂着、看完即忘"，详情页在名言主卡片下方新增 **4 个深度内容板块**，让每条名言都拥有可消化、可落地的解读：

| 板块 | 图标 | 内容 |
|-----|------|------|
| **核心解读** | 🧭 | 用大白话拆解这句话到底在讲什么，避免字面误读 |
| **应用实操** | 🛠 | 3-5 条带编号的可执行清单，把道理翻译成"今天就能做的事" |
| **生动案例** | 📚 | 一段真实的历史故事或场景化案例，带时间、地点、人物和结果 |
| **大师视角** | 💭 | 从该大师整体思想体系出发，给这句话定位；结尾引导跳转大师详情页 |

**三级内容匹配策略（`src/lib/interpretations.ts`）：**

1. **精品匹配（14 条）** — 按 `master_id + 关键字` 精准命中巴菲特、芒格、格雷厄姆、林奇、达利欧、马克斯、博格、卡拉曼、塔勒布等的经典名言，提供**原创的**深度解读 + 实操 + 案例（如"1988 年巴菲特买入可口可乐""LTCM 1998 年覆灭""2008 年《Buy American, I Am.》"等）
2. **标签兜底（10 个）** — 10 个主题标签各对应一份高质量通用解读模板
3. **大师视角兜底（12 位）** — 每位大师预置一段"他整体思想的核心线索"

即：最热门的十几条拿到**独一无二的原创内容**，其他名言也能至少拿到"标签通用解读 + 大师视角"，不会出现空白板块。

## 五、非功能需求

### 5.1 性能

| 指标 | 要求 |
|-----|------|
| 首屏加载 | ≤ 1.5s（国内 CDN 加速） |
| 搜索响应 | ≤ 300ms |
| 并发支持 | ≥ 1000 QPS |

### 5.2 SEO

- 服务端渲染（SSR）/ 动态路由，确保搜索引擎收录
- 每条名言有独立 URL
- 自动生成 sitemap

### 5.3 响应式设计

- 移动端优先设计，适配手机、平板、桌面端

### 5.4 多主题风格切换 ✅

用户可通过导航栏右侧的主题切换器，在 5 种视觉风格之间自由切换，选择自动保存到本地。

| 主题 | 风格 | 视觉特点 |
|-----|------|---------|
| 🌃 **赛博朋克**（默认） | 霓虹未来 | 深黑底色、青紫霓虹、扫描线特效、glitch 文字抖动、等宽字体、尖锐直角、卡片悬停霓虹发光 |
| 💎 经典金 | 温暖雅致 | 金色渐变、暖色调、衬线字体、大圆角 |
| 🖌️ 水墨丹青 | 古朴意韵 | 宣纸色底、棕墨配色、宋体、极小圆角 |
| ◻️ 极简主义 | 纯净克制 | 纯黑白灰、无装饰、系统字体 |
| 🔮 暗夜星辰 | 神秘深邃 | 深紫底色、紫色光晕、衬线字体 |

**技术实现：**
- 通过 `data-theme` 属性 + 40+ CSS 自定义变量驱动全站样式
- `localStorage` 持久化用户选择
- `<head>` 内联脚本防闪烁（FOUC）
- 切换时全站 0.4s 平滑过渡动画
- 赛博朋克主题额外特效：扫描线覆盖、霓虹发光边框、glitch 文字抖动、导航/页脚渐变发光线

### 5.4 内容管理（后期规划）

- 后台管理系统，支持名言/大师信息的增删改查
- 支持批量导入（Excel/CSV）
- 名言审核流程（草稿 → 审核 → 发布）

### 5.5 部署方案 ✅ (v1.2 新增)

项目已支持**子路径部署**，线上版本挂在主站 `/investmentQuotes` 路径下：

| 配置项 | 说明 |
|-------|------|
| `next.config.ts` | 设置 `basePath` / `assetPrefix` 为 `/investmentQuotes`，静态资源与路由自动加前缀 |
| `src/lib/basePath.ts` | 统一的前端路径拼接工具，避免硬编码 |
| `Dockerfile` | Node 20-slim，构建阶段内置腾讯云 apt / npm 镜像，适配 ap-guangzhou 机房加速；运行时持久化 `data/` 目录 |
| 反代 | Nginx 反向代理 `/investmentQuotes` → 容器 `:3000`，与主站共用 80 端口 |
| 容器 | `docker run -d -p 9090:3000 -v /root/investmentQuotes_data:/app/data investment-quotes:vN` |
| 生产地址 | `http://www.luliming.xyz/investmentQuotes` |
